# WG Forge Test task
Works with the WG Forge test task environment.  
Get docker image:  
`docker pull yzh44yzh/wg_forge_backend_env:1.1`  
Run it:  
`docker run -d -p 5432:5432 --name backenv yzh44yzh/wg_forge_backend_env:1.1`

Requirements:  
`python 3.6`  
`psycopg2`
